Introduction

Welcome to the ASP.NET Zero and ABP training documentation. This training is designed to help developers build modular, maintainable, and scalable applications using these powerful frameworks. You will learn everything from foundational concepts to advanced implementation techniques.

ASP.NET Zero Training

Core Topics

  • Getting started with ASP.NET Zero & its architecture
  • Multi-tenancy and modularity concepts
  • Building CRUD applications with EF Core
  • Dependency Injection and Domain-Driven Design

Advanced Practices

  • Customizing themes and layouts
  • Localization and authorization techniques
  • Implementing real-time notifications

Hands-On Project

Participants will build a multi-tenant to-do application, leveraging ASP.NET Zero's modular structure and core features.

ABP.IO Training

Core Topics

  • Introduction to ABP.IO and its architecture
  • Setting up ABP Studio and creating projects
  • Modular development and layered application structure
  • Creating CRUD operations and seeding data

Advanced Practices

  • Custom repositories and UI development (Angular/Blazor)
  • Implementing event-driven architecture with ABP features
  • Logging, caching, and audit logging
  • Localization and object mapping

Hands-On Project

Participants will develop a user management system with ABP.IO, incorporating advanced logging, role management, and localization.

Add-On Modules

  • Test and validate the setup
